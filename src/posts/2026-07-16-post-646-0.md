Proses daftar ni percuma je untuk pelan asas dan senang sangat. Dah ada akaun, terus pergi ke bahagian Workers & Pages. Anda tak perlu pun nak sambungkan domain kalau projek ni cuma nak guna Workers sebagai proksi API.*   **Daftar Akaun**: Layari Cloudflare.com dan daftar. Pilih pelan percuma untuk mula.*   **Buat Worker**: Dekat papan pemuka Cloudflare, cari bahagian "Workers & Pages" dan klik "Create Application" atau "Create a Worker". Anda akan dapat nama subdomain automatik (contohnya, `nama-anda.workers.dev`). Kod ni memang ringkas dan kerjanya cuma ambil permintaan yang masuk, hantar ke API DeepSeek, dan pulangkan respons kepada klien anda. Buka editor Workers anda dan ganti kod sedia ada dengan yang ni:```javascriptaddEventListener('fetch', event => {  event.respondWith(handleRequest(event.request));});async function handleRequest(request) {  const DEEPSEEK_API_BASE = 'https://api.deepseek.com';  const url = new URL(request.url);    // Bina URL permintaan ke DeepSeek API  url.hostname = new URL(DEEPSEEK_API_BASE).hostname;  url.protocol = new URL(DEEPSEEK_API_BASE).protocol;    // Buat permintaan baru ke DeepSeek API  const newRequest = new Request(url.toString(), {    headers: request.headers,    method: request.method,    body: request.body,    redirect: 'follow'  });  // Hantar permintaan dan kembalikan respons  return fetch(newRequest);}```**Penjelasan Kod**: Kod ini 'pintas' setiap permintaan yang dihantar ke Workers anda. Kemudian, ia akan bina semula URL permintaan untuk tuju ke `https://api.deepseek.com` sambil kekalkan semua header, kaedah (GET, POST, dll.), dan badan permintaan asal. Akhir sekali, ia hantar permintaan baru ni ke DeepSeek API dan pulangkan respons balik kepada pemohon asal. Boleh je guna alat macam `curl` dekat terminal anda.**Ujian Tanpa Proksi (Terus ke DeepSeek API):**```bashcurl -s -o /dev/null -w "%{time_total}s\n" https://api.deepseek.com/v1/chat/completions -H "Authorization: Bearer YOUR_DEEPSEEK_API_KEY" -d '{"model": "deepseek-chat", "messages": [{"role": "user", "content": "Hello"}]}'```Gantikan `YOUR_DEEPSEEK_API_KEY` dengan kunci API anda. Ini akan bagi anda masa keseluruhan yang diambil untuk dapat respons terus dari DeepSeek.**Ujian Dengan Proksi Cloudflare Workers:**```bashcurl -s -o /dev/null -w "%{time_total}s\n" https://nama-anda.workers.dev/v1/chat/completions -H "Authorization: Bearer YOUR_DEEPSEEK_API_KEY" -d '{"model": "deepseek-chat", "messages": [{"role": "user", "content": "Hello"}]}'```Gantikan `https://nama-anda.workers.dev` dengan URL Workers anda.
